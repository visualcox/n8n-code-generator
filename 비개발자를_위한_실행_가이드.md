# 🎯 비개발자를 위한 초간단 실행 가이드

## 📌 이 앱은 무엇인가요?

여러분이 원하는 자동화 작업을 말로 설명하면, AI가 자동으로 n8n 워크플로우 코드를 만들어주는 앱입니다!

예를 들어:
- "Gmail에서 중요한 메일이 오면 Slack으로 알림 보내줘"
- "구글 시트에 새 행이 추가되면 Notion에도 추가해줘"

이런 요청을 입력하면 자동으로 완벽한 n8n 코드를 만들어줍니다!

---

## 🎬 실행 방법 (Windows 기준)

### 1단계: Python 설치하기 (5분)

Python은 이 앱의 뒷단을 실행하는데 필요합니다.

1. **Python 다운로드**
   - https://www.python.org/downloads/ 접속
   - 큰 노란색 버튼 "Download Python" 클릭
   - 다운로드된 파일 실행

2. **Python 설치**
   - ⚠️ **중요**: 설치 첫 화면에서 "Add Python to PATH" 체크박스를 **반드시** 체크하세요!
   - "Install Now" 클릭
   - 설치 완료까지 대기

3. **설치 확인**
   - 키보드에서 `Windows 키` + `R` 눌러서 실행창 열기
   - `cmd` 입력하고 Enter
   - 검은 창(명령 프롬프트)이 열리면 다음 입력:
   ```
   python --version
   ```
   - "Python 3.x.x" 같은 메시지가 나오면 성공! ✅

### 2단계: Node.js 설치하기 (5분)

Node.js는 앱의 화면(프론트엔드)을 실행하는데 필요합니다.

1. **Node.js 다운로드**
   - https://nodejs.org/ 접속
   - 왼쪽 버튼 "LTS" 버전 다운로드 클릭
   - 다운로드된 파일 실행

2. **Node.js 설치**
   - 모든 화면에서 "Next" 클릭
   - 설치 완료까지 대기

3. **설치 확인**
   - 다시 `Windows 키` + `R` → `cmd` → Enter
   - 검은 창에 다음 입력:
   ```
   node --version
   ```
   - "v18.x.x" 같은 메시지가 나오면 성공! ✅

### 3단계: OpenAI API 키 준비하기 (10분)

앱이 AI를 사용하려면 OpenAI API 키가 필요합니다.

1. **OpenAI 가입**
   - https://platform.openai.com/ 접속
   - "Sign up" 클릭하여 가입 (Google 계정으로 가능)

2. **API 키 만들기**
   - 로그인 후 https://platform.openai.com/api-keys 접속
   - "Create new secret key" 클릭
   - 이름 입력 (예: "n8n generator") → "Create"
   - **중요**: 나타나는 키를 복사해서 메모장에 저장! (다시 볼 수 없습니다)
   - 예시: `sk-proj-abc123...xyz`

3. **결제 설정** (선택사항)
   - API 키를 사용하려면 소액의 크레딧 충전 필요 ($5~10)
   - https://platform.openai.com/settings/organization/billing 에서 설정

### 4단계: 앱 설치하기 (10분)

이제 앱을 설치하겠습니다!

1. **명령 프롬프트 열기**
   - `Windows 키` + `R` → `cmd` → Enter

2. **앱 폴더로 이동**
   ```
   cd /d C:\Users\사용자이름\webapp
   ```
   (실제 경로로 변경하세요)

3. **Backend(뒷단) 설치**
   ```
   cd backend
   python -m venv venv
   venv\Scripts\activate
   pip install -r requirements.txt
   ```
   - 설치 중 여러 메시지가 나옵니다 - 걱정마세요! 정상입니다.
   - 5~10분 정도 걸립니다 ☕

4. **환경 설정 파일 만들기**
   
   a. 메모장 열기
   
   b. 다음 내용 복사해서 붙여넣기:
   ```
   APP_NAME="n8n JSON Code Generator"
   DEBUG=True
   HOST=0.0.0.0
   PORT=8000
   
   DATABASE_URL=sqlite+aiosqlite:///./n8n_generator.db
   
   DEFAULT_LLM_PROVIDER=openai
   OPENAI_API_KEY=여기에_아까_복사한_API_키_붙여넣기
   
   LEARNING_ENABLED=True
   LEARNING_SCHEDULE_CRON=0 0 * * 0
   N8N_DOCS_URL=https://docs.n8n.io
   N8N_TEMPLATES_URL=https://n8n.io/workflows
   
   CORS_ORIGINS=["http://localhost:3000"]
   ```
   
   c. `여기에_아까_복사한_API_키_붙여넣기` 부분을 실제 API 키로 교체
   
   d. "파일 → 다른 이름으로 저장"
      - 저장 위치: `C:\Users\사용자이름\webapp\backend`
      - 파일 이름: `.env` (점 포함!)
      - 파일 형식: "모든 파일"
   
   e. 저장하고 메모장 닫기

5. **Frontend(화면) 설치**
   
   새 명령 프롬프트 창을 하나 더 열고:
   ```
   cd /d C:\Users\사용자이름\webapp\frontend
   npm install
   ```
   - 5~10분 정도 걸립니다 ☕

### 5단계: 앱 실행하기! 🚀

이제 앱을 실행할 차례입니다!

**명령 프롬프트 창 2개가 필요합니다.**

#### 창 1️⃣: Backend 실행
```
cd /d C:\Users\사용자이름\webapp\backend
venv\Scripts\activate
python main.py
```

다음과 같은 메시지가 나오면 성공! ✅
```
🚀 Starting n8n JSON Code Generator...
✅ Database initialized
✅ Learning scheduler started
🌐 Server running on 0.0.0.0:8000
```

⚠️ **이 창은 닫지 마세요!** (앱이 계속 실행되어야 합니다)

#### 창 2️⃣: Frontend 실행

새 명령 프롬프트 창을 열고:
```
cd /d C:\Users\사용자이름\webapp\frontend
npm run dev
```

다음과 같은 메시지가 나오면 성공! ✅
```
  VITE v5.x.x  ready in xxx ms
  
  ➜  Local:   http://localhost:3000/
```

⚠️ **이 창도 닫지 마세요!**

### 6단계: 앱 사용하기! 🎉

1. **웹 브라우저 열기**
   - Chrome, Edge, Firefox 등 아무거나
   - 주소창에 입력: `http://localhost:3000`
   - Enter!

2. **첫 화면이 나타납니다!** 🎊
   - 왼쪽에 메뉴가 보입니다
   - 홈, 히스토리, 학습 관리, 설정

3. **LLM 설정하기** (처음 한 번만)
   - 왼쪽 메뉴에서 "설정" 클릭
   - "새 LLM 설정 추가" 버튼 클릭
   - 정보 입력:
     ```
     설정 이름: My GPT-4
     제공자: OpenAI
     모델 이름: gpt-4-turbo-preview
     API Key: (아까 복사한 API 키 붙여넣기)
     ```
   - "기본 설정으로 사용" 체크박스 체크 ✅
   - "추가" 버튼 클릭

4. **첫 번째 워크플로우 만들기!**
   - 왼쪽 메뉴에서 "홈" 클릭
   - 큰 입력창에 원하는 자동화 입력:
   
   ```
   매일 아침 9시에 Gmail에서 미읽은 메일을 확인하고,
   중요한 메일이 있으면 Slack으로 알림을 보내주세요.
   ```
   
   - "분석 시작" 버튼 클릭
   - AI가 추가 질문을 합니다 - 답변하세요!
   - 개발요구서가 만들어지면 검토
   - "승인 및 JSON 생성" 클릭
   - 잠시 기다리면... 완성! 🎉

5. **생성된 코드 사용하기**
   - "복사" 버튼으로 코드 복사
   - n8n 앱 열기
   - "Import from JSON" 클릭
   - 붙여넣기
   - 완료! 🎊

## 🆘 문제 해결

### "python을 찾을 수 없습니다" 오류
→ Python 설치 시 "Add Python to PATH" 체크 안 함
→ Python 재설치 (체크박스 꼭 체크!)

### "npm을 찾을 수 없습니다" 오류
→ Node.js가 제대로 설치 안 됨
→ Node.js 재설치

### Backend가 실행 안 됨
→ `.env` 파일이 올바른 위치에 있는지 확인
→ API 키가 올바른지 확인

### Frontend 화면이 안 나옴
→ Backend가 실행 중인지 확인 (창 1️⃣)
→ 주소가 `http://localhost:3000`인지 확인

### API 키 오류
→ OpenAI 계정에 크레딧이 있는지 확인
→ API 키를 다시 생성해서 입력

```

발신자가 boss@company.com 이거나 제목에 [긴급]이 포함된

메일 제목과 발신자 정보를 알림으로 보내주세요.
```

```

```

